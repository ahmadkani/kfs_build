# Git Filesystem Manager (KFS)

An easy filesystem abstraction for working with Git-backed storage inside web apps, with Service Worker support!

---

## Features
- ✅ Mount and unmount in-memory Git filesystem
- ✅ Read, write, and delete files easily
- ✅ Service Worker support for offline sync
- ✅ Auto versioning and conflict resolution

---

## Installation

```bash
npm install kfs
